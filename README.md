# Selenium Test Automation

This project is a Selenium + Pytest test automation framework for SauceDemo.

## Technologies

- Python
- Selenium
- Pytest
- Pytest HTML
- GitHub Actions
- Page Object Model

## Test Scenarios

- Successful login
- Invalid login
- Add product to cart
- Remove product from cart
- Multiple products cart test
- Checkout flow
- Invalid checkout validation
- Logout

## Run Tests

```bash
pytest -v
